Hardware

Three-Module System

Gait Speed Module

Sensor: 2D LiDAR (10Hz)
Range: Up to 10m customizable walking distance
Protocols: Static and dynamic start conditions
Output: Velocity profile, acceleration, gait metrics

Sit-to-Stand Module

Detection: Multi-modal sensor array
Protocols: 5-repetition or 30-second tests
Tracking: Seat height, standing time per cycle
Output: Total time, rhythm analysis

Balance Module

Platform: Pressure-sensitive footplate
Positions: Side-by-side, semi-tandem, tandem
Tracking: Center of pressure, sway metrics
Output: Hold time, stability score
